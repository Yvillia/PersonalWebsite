  • Calculation-Assistant:
    • Summary: An app designed to help with mathematics as a mobile app. Basically the app allows one to type in an algebraic expression that will then show steps on how to simplify the expression into simplest form by using a Binary Expression Tree to produce a thorough solution. The app also supports differentiation and integration via the use of Wolfram Alpha's API, but these techniques do not show steps yet.

  • ASL-Recognition:
    • Summary: A Computer Science group project focused on developing a Machine Learning Model that can accurately classify fingerspelling in American Sign Language from an input image. The project required the development of custom dataloaders and a custom architecture based off of research papers. The model was developed using Pytorch and was capable of reaching 93% testing accuracy although real-time analysis requires much more training. The model was hooked up to a website that was developed using a python tool called Flask.
    • Languages and Skills: Python, Pytorch, Machine Learning, Flask, HTML & CSS, Fullstack-Development
    • Project Link

  • Cerberus:
    • Summary: A cybersecurity application currently being developed with another co-developer with the hopes of starting a business. The project uses one-way password encryption that is maintained locally as to prevent insecurity that comes with having information maintained in a common database. The app allows for many accessibility options and also for new passwords to be created while only needing to remember a single master password.
    • Languages and Skills: ElectronJS, NodeJS, HTML & CSS, Encryption and Security, Fullstack-Development
    • Project is Currently in Progress and is Private
